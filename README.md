## Quick Start

Add `nuance-ui` to the `modules` section of your `nuxt.config.ts`:

```typescript
export default defineNuxtConfig({
	modules: ['nuance-ui'],
})
```

Now you can use components in your application:

```vue
<template>
	<div>
		<NButton variant='primary'>
			Click me
		</NButton>

		<NCard>
			<p>Card content goes here</p>
		</NCard>
	</div>
</template>
```
